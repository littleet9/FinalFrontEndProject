# Weather Keeper


## Project setup
Ok so make sure you cd into finalfrontendproject-main. The same directory with src, public, ...etc.
Then do:
```
npm install -g vue
npm install -g @vue/cli
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Runs backend
```
npm run backend
```
